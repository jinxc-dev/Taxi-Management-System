# Taxi-Management-System
### Laravel-Vue project
- laravel framework.
- Vue js.
- FCM server for messaging to multi devices.
- Vue2-Google-Map.
**![Imgur](https://github.com/timegold-websrc/Taxi-Management-System/blob/master/screen.png)**
### installation
- composer install.
- php artisan key:generate
- php artisan migrate
- php artisan serve
- npm install
- npm run dev
